About the Project

Simplee Supplements is a frontend landing page project built for a health and wellness brand. The goal was to create a visually compelling, conversion-focused page that clearly communicates the brand's products and value proposition.

The design prioritizes clean typography, strong product visuals, and a smooth user experience across all screen sizes — from mobile to desktop.

Key Features

  • Hero section with strong CTA and product highlight
  • Product showcase with clean card-based layout
  • Benefits and features section with icon highlights
  • Testimonials / social proof section
  • Fully responsive, mobile-first design
  • Smooth scroll animations and hover interactions
